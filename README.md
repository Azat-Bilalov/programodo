# programodo
Приложения компании по разработке и внедрению открытого ПО. Курсовая работа по БД за 2 курс.

##  Технологии
Фронтенд: React + RTK + Tailwind CSS
Бекенд: Node + Express + pg-promise
БД: PostgreSQL

## Использование
Запуск сервера
```bash
$ cd server
$ npm i
$ npm start
```

Запуск Tauri-приложения
```bash
$ npm i
$ npm run tauri dev
```

Сборка Tauri-приложения:
```bash
$ npm run tauri build
```
